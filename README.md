## Installation

```bash
$ yarn install
```

## Running the app

```bash
# docker-compose
$ docker-compose up -d

# development
$ yarn run start user
$ yarn run start

# watch mode
$ yarn run start:dev user
$ yarn run start:dev

# production mode
$ yarn run start:prod user
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
